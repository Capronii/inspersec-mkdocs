Aula Sans#

Como nosso curso introdutorio se baseia no curso da Sans, vamos disponibilizar o material utilizado no curso para que os integrantes possam consultar sempre que necessario. link para o material: https://www.sans.org/cyberaces/

Capitulo Basico de Linux#

Como um complemento para o curso introdutorio, vamos disponibilizar um capitulo basico de linux, para que os integrantes possam se familiarizar com o sistema operacional. Esse material possui muitos comandos e dicas que podem ser uteis no dia a dia da entidade. link para o download do material: Capitulo basico linux
